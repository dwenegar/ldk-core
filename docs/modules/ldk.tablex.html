aggregate(t[, acc], f)
Applies an accumulator function over a table.

Parameters:

  • t table the table to aggregate over.
  • acc the initial value of the accumulator. (optional)
  • f accumulator the accumulator function to be applied to each key-value pair.

Returns:

    the final accumulator value.
all(t, p)
Determines whether all the key-value pairs of a table satisfy a condition.

Parameters:

  • t table a table containing the key-value pairs to apply the predicate to.
  • p predicate a function to test each key-value pair for a condition.

Returns:

    boolean true if every key-value pair of the table satisfies the specified predicate, or if the table is empty; otherwise false.
any(t, p)

Returns:

    boolean true if any key-value pair of the table satisfies the specified predicate, or if the table is empty; otherwise false.

Calculates the average of the values obtained by applying a transform function to each key-value pair of a table.

  • t table a table to calculate the average of.
  • f transform a transform function to apply to each element. (optional)

Returns:

    number the average of the projected values; nil if the table is empty.

    boolean true if the table contains the specified key; false otherwise.

Compares two tables for equality.

  • t2 table the second table to compare.

Returns:

Groups the key-value pairs of a table according to a specified key selector.

  • f keyselector a function to extract the key for each key-value pair.

Creates a table with the keys of a table.

Parameters:

  • cmp comparator a function used to compare each key. (optional)

Returns:

    table a table with the keys of the input table.
map(t, f)
Projects each key-value pair of a table into a new value.

Parameters:

  • t table a table to invoke the transforma function on.
  • f transform a transform function to apply to each key-value pair; the second parameter is the index of the key-value pair.

Returns:

    table a table whose key-value pairs are the the result of applying the specified transform function on the key-value pairs of the input table.
max(t, f)
Applies a transform function to each value of a table and return the the maximum of the projected values.

Parameters:

  • t table an table to determine the maximum value of.
  • f transform a transform function to apply to each value.

Returns:

  1. the key of maximum projected value in the table, or nil if the table is empty.
  2. the maximum projected value in the table, or nil if the table is empty.
maxby(t, f)
Applies a transform function to each value of a table and returns the maximum according to the projected values.

Parameters:

  • t table a table to determine the maximum value of.
  • f transform a transform function to apply to each value.

Returns:

  1. the key of maximum value in the table according to the transform function or nil if the table is empty.
  2. the maximum value in the table according to the transform function, or nil if the table is empty.
